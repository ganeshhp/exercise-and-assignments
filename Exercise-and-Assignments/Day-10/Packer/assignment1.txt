### write packer configuration file for creating AWS AMI using the standard configuration file contents like, packer block, required_plugins, variable, sources, build, provisioner, conditional statement for selecting a particular build for provision, Post-processors.

### for AMI, we have to authenticate on AWS environment using AWS Secret_key and AWS Secret_Access_key. These will have to be set as environment variables on the packer machine. and the same will then help in authenticating packer build on AWS environment.
